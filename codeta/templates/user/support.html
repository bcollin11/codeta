FAQ

What is Code TA?

Code TA is a web-based tool meant to automatically grade programming assignments for professors. To learn more, visit the about page.

How do I get started?

First create an account on the Code TA website by clicking here or pressing Join on the menu bar. After joining and logging in, visit your courses page to add programming courses. From each course page add assignments which need to be graded. For each assignment, you can upload unit tests and student code to be evaluated. A report will be generated with full details on tests passed.

How do I grade an assignment?

After creating the appropriate course and assignment, upload both the unit tests and student codde.

I've encountered an error. What do I do?

Please submit a description of the error with the time encountered and a screenshot if possible to one of the team members. Member information can be found on the contact page.
